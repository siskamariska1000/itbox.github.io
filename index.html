About Web Author

ahyana rizky pratama

Salam kenal, nama saya Ahyana Rizky Pratama, biasa dipanggil Ariz

Saat ini saya bekerja sebagai seorang engineer di perusahaan kelapa sawit yang beroperasi di daerah blok M. Saya merupakan lulusan Teknik Mesin Universitas Indonesia. Saya tertarik untuk mempelajari ilmu developer karena saya liat saat ini banyak startup ataupun perusahaan-perusahaan yang menggunakan ilmu computer science ini sebagai major bisnis mereka. Di sisi lain saya lihat masih banyak permintaan dari berbagai perusahaan yang bergerak di bidang teknologi terhadap developer yang mumpuni. Mumpung masih muda, saya tertarik untuk mengambil peluang ini.

Alasan lain saya mau belajar programming dan menjadi developer adalah saya ingin punya nilai tambah dan nilai jual yg berbeda dari kebanyakan orang. Dari sekian banyak pilihan akhirnya suatu hari dapat info mengenai Hacktiv8 dari facebook ads, setelah itu saya coba melihat program yang ditawarkan dan langsung memutuskan untuk bergabung bersama keluarga besar Hacktiv8 untuk menjadi seorang developer professional.
